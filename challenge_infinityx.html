Biggest Challenges

Even though I loved working on this project, I did experience some challenging obstacles that I had to deal with

CSS Styling

One of my biggest challenge of this whole project was styling it using raw CSS as some of the animations and containers took a lot of time to get correct

Structuring New Features

Another one of my biggest challenges was creating components for new features not in the original SpaceX site that still kept the same styling as my original pages even though I had no styling reference for such components

Smooth Rerendering

At first I was having some trouble getting all of my desired components to rerender smoothly when the appropriate slice of state goes through a change, but then I figured out how to by chaining promises together
