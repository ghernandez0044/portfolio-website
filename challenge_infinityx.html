Biggest Challenges

Even though I loved working on this project, I did experience some challenging obstacles that I had to deal with

CSS Styling

A significant obstacle encountered throughout this project involved the application of raw CSS for styling, particularly in relation to intricate animations and containers that demanded considerable time and effort to achieve the desired accuracy.

Structuring New Features

One of my significant hurdles involved designing components for new features that were not present in the original SpaceX website. Despite lacking styling references for these components, it was crucial to maintain consistency with the styling of my original pages.

Smooth Rerendering

Initially, I encountered difficulties achieving seamless rerendering of my desired components when the relevant state slice undergoes modification. However, I subsequently devised a solution by leveraging promise chaining.
