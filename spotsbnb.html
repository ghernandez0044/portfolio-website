Overview

Components include the Backend API server and database (Express-Sequelize) to handle data, requests, and responses as well as a Frontend (React) to display data on a front-end application. Technologies consist of JavaScript, SQL, Sequelize, Express, PostgreSQL, React and Redux. Worked with waterfall methodology on this project concluding the backend server first using Express and Sequelize and then finalizing the frontend application using React.
